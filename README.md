# EveeDiscordBot

- Requirements:

	- Discord.js / Rest / Discord API Types / Builders

 	```
 	npm install discord.js @discordjs/rest discord-api-types @discordjs/builders
 	```
	
	- dotenv

	```
 	npm install dotenv
 	```

	- Images-Scraper
	
	```
	npm install images-scraper
	```
